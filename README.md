Ren is framemork(inspired by Ren'Py) for story driven games in Godot.

## Story driven games

It means games that are for example:

- Visual Novels (WIP):
	- GDScript (almost done)
	- VisualScript (WIP)
	- RenScript (ToDo)
- Click & Point Adventures (ToDo)
- RPG(ToDo)
